# Changelog

All notable changes to this template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Template Release

### Added

- Modern TypeScript monorepo structure with Turborepo
- Next.js 16 web application with App Router
- Elysia backend server with tRPC integration
- PostgreSQL database with Drizzle ORM
- Better-Auth authentication system
- TailwindCSS 4 with shadcn/ui components
- Comprehensive documentation (README, TEMPLATE_SETUP, CONTRIBUTING)
- GitHub templates for issues and pull requests
- Environment variable templates (.env.example files)
- MIT License
- Node.js version specification (.nvmrc)

### Documentation

- Detailed setup instructions
- Deployment guides for Vercel and Railway/Render
- Contributing guidelines
- Template customization guide
- Tech stack documentation

### Developer Experience

- Type-safe APIs with tRPC
- Hot reload for development
- Database migrations with Drizzle
- Monorepo build optimization with Turborepo
- Consistent code style configuration
